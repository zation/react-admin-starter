# React Admin Starter

### libraries

* React
* Redux
* React Router
* Lodash/fp
* Recompose
* Redux Form
* Ant Design
* Recharts
* Webpack
* React Hot Loader
* Faker

## Installation

```bash
npm install
```

## Development

Start mock server

```bash
npm run mocker
```

Start dev server

```bash
npm start
```
